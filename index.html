What makes it tick

Engineering details that matter

🧠 LSTM with early stopping

A 2-layer LSTM (hidden size 128, dropout 0.2) trained with Adam, a ReduceLROnPlateau scheduler, and early stopping on validation loss — so it generalises instead of memorising.

⚙️ 12 engineered indicators

SMA, EMA, MACD & signal, RSI, Bollinger Bands, ATR, OBV, and ROC — each computed with pure pandas/NumPy and documented in the README.

🎛️ No-code parameter tuning

The Dash dashboard lets anyone pick a ticker and adjust the input window and forecast horizon with sliders — zero code changes required.

🧹 Robust data cleaning

IQR-based anomaly detection, short-gap forward-filling, and removal of >15% single-day jumps that usually signal an unadjusted split or data error.

Get running

Up and forecasting in six commands

# 1 · clone the repo
$ git clone https://github.com/nakwafurkhan/stocksage-lstm.git
$ cd stocksage-lstm
# 2 · install pinned dependencies
$ pip install -r requirements.txt
# 3 · download data & build features
$ python -m src.data_loader
$ python -m src.preprocessing
# 4 · train a ticker
$ python src/train.py --ticker AAPL
# 5 · launch the dashboard → http://localhost:8050
$ python app/dashboard.py
